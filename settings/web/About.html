About

The Regio Vinco Map Maker was conceived of, designed, and implemented by Richard McKenna. The application was built in order to create playable maps for Regio Vinco ©, a geography game for learning about the world. Note this application employs use of the PropertiesManager, jTPS, and Desktop Java Framework libraries, which were also created by and maintained by Richard McKenna.
